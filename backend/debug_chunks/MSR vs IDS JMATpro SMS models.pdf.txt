is to provide quantitative analysis of the difference between the output of the solidification model (Chill) run with the same casting parameters but with steel grade properties calculated by three different models: IDS, JMatPro an equilibrium. At the moment: - Chill uses steel grade features from IDS. - CCD uses steel grade features from JMP, except for the Conductivity which is coming from IDS. The different features contribute to the deviation of the metallurgical length calculated with the two solidification models. In general, even with the same input (CCM design, casting parameters) the choice of different
